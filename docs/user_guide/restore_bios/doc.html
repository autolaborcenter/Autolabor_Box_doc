恢复 BIOS¶

如果你不小心更改了主板 BIOS 配置，本文将引导你如何恢复主板 BIOS 到最佳设置。

准备¶

  • 存有主板 BIOS 配置文件的 Autolabor OS 镜像盘 *1

开始¶

主板 BIOS 配置文件可以联系售后客服获取

常见问题¶

  • “这些操作会导致我的代码/文档/程序丢失吗？”

    不会。

  • “我可以其他电脑（非 Autolabor PC）上执行这些操作吗”

    你拥有操作的自由，但我们强烈不建议这样做。
