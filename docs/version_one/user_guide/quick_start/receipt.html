收货指南¶

检查¶

收到货后请不要急于签收，先检查外包装有无破损，如包装破损或贴于外包装上的防震标签感应器显示为红色，请拍照为证，并与快递员一同开箱验货，待确认无误后再签收，如需帮助可联系Autolabor客户服务平台。

清点¶

收件后打开包装，对照产品装箱单进行清点，如有短缺请于三日内联系客服，过期不补。
