# 注意事项

* 此次操作将恢复AutolaborPC主板到默认设置，但不会影响你存储在硬盘上的文稿数据、应用程序、隐私信息、使用记录等

* 此操作仅支持在AutolaborPC上进行，请勿在其他工控机/PC上尝试

* 即便你非常了解主板设置的相关内容，仍然不建议你任意修改主板设置，由此导致设备损坏的情况我们无法提供相关的保修服务
